Development Projects

MiniCAML: a DIY Programming Language

Implemented a programming language called MiniCAML in OCaml to achieve available functionalities such as free variables, substitution, evaluation, type checking, and type inference. MiniCAML has been extended to n-ary function abstraction, application, and recursion.

Learn more

Interactive 3D Open World Games

A game where a helicopter can fly in the 3D space freely. Used the Perlin noise to randomly create the topography (infinite & never repeat), including trees, mountains, rivers, and cloud. Used Gerstner wave function to simulate water fluctuations. Energy decreased with flight time, and the aviator needed to search for randomly generated energy balls to keep the helicopter flying.

Research Projects

Self-Supervised Learning

Focused on aligning gradients between different domains. The work revealed that gradient variance matching was particularly effective, outperforming a widely used method Inter-domain Gradient Matching by 5% in accuracy on the DomainBed dataset.

Learn more

Trustworthy Machine Learning

Aimed at addressing challenges of class imbalance, label noise, and data scarcity. To tackle these issues, a method that adjusts training difficulty was introduced. It involved adding perturbations to model parameters before updating them.

Learn more

COVID-19 Transmission Modelling

Built a mathematical model focusing on symptom onset, which is calibrated using data from severe outbreaks. Analyzed the impact of different disease control policies in various areas, observing the reduction of basic reproduction number R0.
